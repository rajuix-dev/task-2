# 📝 To-Do List Web App (Vanilla JavaScript)

## 📌 Project Overview

This is a simple and interactive **To-Do List Web Application** built using **HTML, CSS, and Vanilla JavaScript**.
The app allows users to **add, complete, and delete tasks dynamically** without reloading the page.

---

## 🎯 Objective

To learn and implement:

* DOM Manipulation
* Event Handling
* Dynamic UI Updates using JavaScript

---

## 🛠️ Tools Used

* VS Code
* Chrome Browser
* Live Server

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* ⚡ Instant UI updates (no page reload)
* ⌨️ Add task using **Enter key**

---

## 📂 Project Structure

```
To-Do-App/
│
├── index.html   # Structure of the app
├── style.css    # Styling of the app
└── script.js    # Functionality (JavaScript)
```

---

## ⚙️ How It Works

1. User enters a task in the input field
2. Clicks the **Add Task** button (or presses Enter)
3. Task is added dynamically to the list
4. Clicking on a task marks it as completed
5. Clicking the delete button removes the task
